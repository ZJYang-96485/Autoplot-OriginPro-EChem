Axis Control Concept

The main X scale controls where both top time ticks and bottom step labels are placed.
  • Main X Scale / Top Axis: real X coordinate, such as time.
  • Bottom Step Labels: labels placed at positions on the same main X scale.
  • Custom label positions: use main X units, such as minutes.
